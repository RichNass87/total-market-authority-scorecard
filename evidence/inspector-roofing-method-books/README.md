## Claim Rules

Use basic sourced claims only:

- instance of
- author or creator
- developer if software/source-code related
- publication date
- language of work or name
- official website
- described at URL
- DOI
- ISBN-13 where confirmed
- source code repository URL

Do not add:

- awards not independently verified,
- registered trademark status for pending applications,
- claims of certification or legal authority,
- unsourced business performance claims,
- promotional descriptions.
