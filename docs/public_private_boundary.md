# Public / Private Boundary

The Total Market Authority Scorecard™ publishes a public educational framework.

It explains:
- The main authority categories
- Public scoring logic
- General score interpretation
- Checklist questions
- Glossary terms
- Example use cases
- Educational methodology
- Limitations

It does not publish:
- Private consulting implementation order
- Internal scoring formulas used in paid work
- Private client audit process
- Proprietary prompt libraries
- Exact content-routing system
- Exact link strategy
- Conversion scoring rules
- Internal market takeover sequencing
- Vendor process
- Private operating checklists

The public framework helps business owners understand how to think.

The private system is how the framework is implemented in real markets.
